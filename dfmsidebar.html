Detailed Description

DFMSideBar is the left sidebar widget of Deepin File Manager

DFMSideBar is the left sidebar widget of Deepin File Manager, provide the interface to manage left sidebar state. Sidebar holds some DFMSideBarItemGroup s which hold their DFMSideBarItem s, and DFMSideBar provide interface to access and manage sidebar items without touching the DFMSideBarItemGroup internal class.

See also DFMSideBarItem.
